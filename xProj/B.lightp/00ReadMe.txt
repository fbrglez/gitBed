B.lightp is a sandbox created to solve the lights-out problem

the essential directories are

bin/  ... executables and source codes from elsewhere

xBin/ ... executable and source code created for this sandbox
          (may have links libraries under xBed as well local libs)

pLib/    ... python library directory

xBenchm/ ... benchmark directory

xLib/    ... tcl library directory

xResults/  ... archives results of experiments and reports/articles
               associated with these experiments

xWork/     ... directory where all invocations are to take place.
               It also contains few transcripts of example invocations

NOTE: the symbolic links point to private directories related to 
      this sandbox and may not be accessible
